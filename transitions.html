Transiciones

Las transiciones de CSS permiten controlar la velocidad de cambio de animación al cambiar propiedades CSS. Los cambios de un estado a otro se dan a través del un período de tiempo. El intervalo de tiempo en que se da la transición es personalizable.

Ejemplo

Código html <div class="wrap-prop"> <div class="container-prop"> <h5>Sin transición</p> <div class="box1-prop box-prop"></div> </div> <div class="container-prop"> <h5>Con transición</p> <div class="box2-prop box-prop"></div> </div> </div> Código css .wrap-prop { margin: 50px; } .container-prop { display: inline-block; width: 300px; } h5 { color: lightgray; font-family: lato; font-size: 20px; font-weight: 200; padding: 20px; text-align: center; text-transform: uppercase; } .box-prop { border-radius: 5px; height: 40px; margin: 50px auto; width: 80px; } .wrap-prop:hover .box-prop { transform: scale(2); } .box1-prop { background: mediumturquoise; } .box2-prop { background: #2b3f53; transition: all 1s; }
Sin transición

Con transición
Las transiciones permiten indicar:

  • transition-property:

    Las propiedades a animar

  • transition-delay:

    El momento en el que inicia la animación

    Ejemplo

    Código html <div class="container-delay"> <div class="box1-delay box-delay"></div> <div class="box2-delay box-delay"></div> <div class="box3-delay box-delay"></div> </div>     		Código css container-delay { margin: 100px; } .box-delay { background: steelblue; border-radius: 5px; display: inline-block; height: 80px; margin: 20px; width: 80px; } .container-delay:hover .box-delay { transform: scale(1.4); } .box1-delay { transition: all 1.3s; } .box2-delay { transition: all 1.3s 0.5s; } .box3-delay { transition: all 1.3s 1s; }

  • transition-duration:

    El tiempo de duración de la animación

    Ejemplo

    Código html <div class="wrap-duration"> <div class="container-duration"> <h5>300ms</p> <div class="box1 box"></div> </div> <div class="container-duration"> <h5>1s</p> <div class="box2 box"></div> </div> <div class="container-duration"> <h5>3s</p> <div class="box3 box"></div> </div> </div> Código CSS .wrap-duration { margin: 50px; } .container-duration { display: inline-block; width: 150px; } h5 { color: lightgray; font-family: lato; font-size: 20px; font-weight: 200; padding: 20px; text-align: center; } .box { border-radius: 50%; height: 40px; margin: 50px auto; width: 40px; } .wrap-duration:hover .box { transform: scale(2); } .box1 { background: #60D4C8; transition: all 300ms; } .box2 { background: #46BAAF; transition: all 1s; } .box3 { background: #3EA69B; transition: all 3s; }
    300ms

    1s

    3s

  • transition-timing-function

    : La manera en que se va a dar la transición de un estado a otro
    • ease: efecto que empieza despacio, va más rápido y luego vuelve a ir despacio
    • linear: efecto que tiene la misma velocidad desde el inicio hasta el final
    • ease-in: efecto de transición que empieza despacio
    • ease-out: efecto de transición que termina despacio
    • ease-in-out: efecto de transición que comienza y termina despacio

      • Ejemplo


      Código html <div class="exampletransitionb"> <div class="ease">ease</div> <div class="linear">linear</div> <div class="easein">ease-in</div> <div class="easeout">ease-out</div> <div class="easeinout">ease-in-out</div> </div> Código CSS .exampletransitionb:hover div{ width: 500px; } div.ease { -webkit-transition: 3s ease; -moz-transition: 3s ease; -o-transition: 3s ease; -ms-transition: 3s ease; transition: 3s ease; } div.linear { -webkit-transition: 3s linear; -moz-transition: 3s linear; -o-transition: 3s linear; -ms-transition: 3s linear; transition: 3s linear; } div.easein { -webkit-transition: 3s ease-in; -moz-transition: 3s ease-in; -o-transition: 3s ease-in; -ms-transition: 3s ease-in; transition: 3s ease-in; } div.easeout { -webkit-transition: 3s ease-out; -moz-transition: 3s ease-out; -o-transition: 3s ease-out; -ms-transition: 3s ease-out; transition: 3s ease-out; } div.easeinout { -webkit-transition: 3s ease-in-out; -moz-transition: 3s ease-in-out; -o-transition: 3s ease-in-out; -ms-transition: 3s ease-in-out; transition: 3s ease-in-out; }

    Ejemplos de transiciones simples

    • Difuminado

      Código html <div class="fade"> <h3>Difuminado</h3> El efecto de difuminado requiere la definición de un estado inicial y un cambio. </div> Código CSS .fade { opacity: 0.5; } .fade:hover { opacity: 1; }

      Difuminado

      El efecto de difuminado requiere la definición de un estado inicial y un cambio.

    • Cambio de color

      • Código html <div class="color"> <h3>Cambio de color </h3> Al darse un cambio de estado se cambia el color. </div> Código CSS .color:hover { background-color: #3EA69B; }

      Cambio de color

      Al darse un cambio de estado se cambia el color.

    • Cambio de tamaño

      • Código html <h3>Cambio de tamaño</h3> Se cambia el ancho y alto del elemento para aumentar o reducir su tamaño. <div class="grow"> Aumenta tamaño. </div> <div class="shrink"> Reduce tamaño. </div> Código CSS .grow:hover { -webkit-transform: scale(1.5); -ms-transform: scale(1.5); transform: scale(1.5); } .shrink:hover { -webkit-transform: scale(0.8); -ms-transform: scale(0.8); transform: scale(0.8); }

      Cambio de tamaño

      Se cambia el ancho y alto del elemento para aumentar o reducir su tamaño.
      Aumenta tamaño.
      Reduce tamaño.

    • Rotar elemento

      • Código html <div class="rotate"> <h3>Rotar elemento</h3> </div> Código CSS .rotate:hover { -webkit-transform: rotateZ(-30deg); -ms-transform: rotateZ(-30deg); transform: rotateZ(-30deg); }
